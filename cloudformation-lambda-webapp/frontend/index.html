⚙️ API エンドポイントの設定

CloudFormation スタックのデプロイ後、コンソールの「出力」タブに表示される ApiEndpoint の値をコピーして貼り付けてください。

💡 例: https://abc12345.execute-api.ap-northeast-1.amazonaws.com/prod
末尾のスラッシュは不要です。
