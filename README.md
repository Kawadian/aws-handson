# aws-handson
AWSの様々なサービスをハンズオン的に利用するためのリポジトリ

## ハンズオン一覧
- [cloudfront-s3-ec2](./cloudfront-s3-ec2/README.md): CloudFront で S3 のキャッシュと EC2 の動的レスポンスを体験するハンズオン
- [cloudformation-lambda-webapp](./cloudformation-lambda-webapp/README.md): CloudFormation で Cognito + Aurora Serverless v2 + API Gateway + Lambda + CloudFront + S3 + SQS + SNS + DynamoDB の会員制タスク管理アプリを一括デプロイするハンズオン（53 リソース）
