# SNS ハンズオン

トピックとサブスクリプションを作成し、通知配信の基本を体験する教材です。

## ゴール

- SNS トピックを作成する
- メールサブスクリプションを追加する
- メッセージ publish で通知が届くことを確認する

## 作るもの

- トピック名: `handson-task-events`
- サブスクリプション: 自分のメールアドレス

## 手順

### 1. トピックを作成する

1. SNS コンソールを開く
2. **トピック > トピックの作成** を選ぶ
3. タイプは **Standard** にする
4. 名前を `handson-task-events` にして作成する

### 2. サブスクリプションを追加する

1. 作成したトピックを開く
2. **サブスクリプションの作成** を選ぶ
3. プロトコルは **Email** を選ぶ
4. 自分のメールアドレスを入力して作成する
5. 届いた確認メールのリンクを開いて承認する

### 3. メッセージを publish する

件名と本文を入力して送信します。例:

- 件名: `Task Created`
- 本文: `user-001 created task-001`

メールが届けば成功です。

## 発展課題

- SQS をサブスクライバとして追加する
- CloudWatch Alarm の通知先として使う

## 片付け

- サブスクリプションを削除する
- トピック `handson-task-events` を削除する

## 学べるポイント

- SNS は「1 回の publish を複数の購読先へ配る」仕組み
- メール、SQS、Lambda など用途に応じて受け側を変えられる
- 非同期イベント連携の出発点として理解しておくと便利
